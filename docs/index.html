Kimibot

The AI that actually does things.

Native Kimi integration for WhatsApp, Telegram, Slack, Discord, and more. Runs on your own infrastructure.

Install
$ npm install -g kimibot@latest
$ kimibot onboard --install-daemon

Quick Start

  1. Install — npm install -g kimibot@latest
  2. Configure — kimibot onboard --install-daemon
  3. Run — kimibot gateway --port 18789
  4. Chat — kimibot agent --message "Deploy prod"

Features

Kimi Native

First-class support for Kimi K2.5, K2, and Moonshot models with 256K context.

Multi-Channel

WhatsApp, Telegram, Slack, Discord, Signal, iMessage, Teams.

Voice & Canvas

Voice wake on macOS/iOS/Android. Live Canvas workspace.

Browser

Built-in Chrome automation. Browse, click, extract data.

Channels

WhatsApp Telegram Slack Discord Signal iMessage Teams WebChat
